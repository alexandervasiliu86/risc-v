risc-v abi

syscalls
om syscalls te gebruiken, gebruiken we register a7 wat een argument register is (a0-a7)
a0 is voor file descriptors
a1 is buffer pointer
a2 is lengte
a7 is syscall id
we kunnen a3, a4,a5 en a6 gebruiken voor extra argumenten indien syscall het nodig heeft

dan hebben we ook tijdelijke (temporary registers)(t0-t6)

word dus door de caller gebruikt (wordt niet bewaard na function calls)
dus we gebruiken dit als het niet van belang is dat die waarde blijft ook na function calls
daarna hebben we ook saved registers (s0-s11)
deze registers blijven hetzelfde ook na function calls en het kan ook gebruikt worden in function calls


dan hebben we zero register(x0)
zero register blijft altijd 0 en kan dus niet aangepast worden dus read-only,
